# Screenshot Placeholders

This directory should contain the following screenshots. Use the descriptions below to capture them.

## Required Screenshots

### dashboard.png
**Description**: Main dashboard showing all panels - metrics, persistence diagram, transaction graph, and alerts
**Size**: 1920x1080 minimum
**What to capture**: Full browser window with TopoGuard dashboard loaded

### dashboard-overview.png
**Description**: Full dashboard view with all components visible
**Size**: 1920x1080 minimum
**What to capture**: Complete dashboard layout

### persistence-diagram.png
**Description**: 3D interactive persistence diagram showing birth-death pairs
**Size**: 1200x800 minimum
**What to capture**: Close-up of the persistence diagram visualization

### transaction-graph.png
**Description**: Network graph showing transaction flows between accounts
**Size**: 1200x800 minimum
**What to capture**: Transaction network visualization with highlighted anomalies

### alerts-panel.png
**Description**: Recent alerts panel showing fraud detection results
**Size**: 800x600 minimum
**What to capture**: Alert cards with explanations

### transaction-details.png
**Description**: Transaction detail modal with full analysis
**Size**: 1000x700 minimum
**What to capture**: Modal dialog showing transaction breakdown

### api-docs.png
**Description**: Interactive API documentation (Swagger UI)
**Size**: 1920x1080 minimum
**What to capture**: API documentation page

## How to Add Screenshots

1. Capture screenshots using browser dev tools or screenshot tools
2. Save them in this directory with the exact names above
3. The README will automatically display them
